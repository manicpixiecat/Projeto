<h1># ProjetoIndividual</h1>

🚀 Começo do Projeto
Iniciei o projeto fazendo uma separação de paleta de cores e formato de caixa que gostaria na página inicial, imagine que gostaria que minha foto fosse redirecionada para o meu perfil do linkin, que cada página fosse separada em tópicos e que houvesse alguma interação com o usúario usando a tag hover.
